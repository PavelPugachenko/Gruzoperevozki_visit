#  одностраничник

Лендинг-визитка транспортной компании :
- `index.html` — структура и контент страницы.
- `styles.css` — адаптивный дизайн, анимации, мобильное меню.
- `script.js` — интерактив: бургер-меню, reveal-анимации, форма в Telegram.

## запуск

```bash
python3 -m http.server 8080
```

После этого сайт будет доступен по адресу `http://localhost:8080`.
